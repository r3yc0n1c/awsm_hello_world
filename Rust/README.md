# [Rust](https://www.rust-lang.org/)
> 2024

## Setup

```sh
sudo apt update
sudo apt install build-essential
curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh
source $HOME/.cargo/env
rustc --version
```

## Run

```sh
cargo run

     Running `target/debug/Rust`
Hello World!
```
